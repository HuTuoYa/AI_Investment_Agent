## 事件1
- 事件标题：Alphabet拟融资800亿美元用于扩张AI基础设施与全球算力
- 发生时间：2026-06-01
- 涉及公司 / 产业：Alphabet / Google Cloud / AI数据中心 / 算力基础设施
- 事实摘要：Alphabet官方公告拟进行总额800亿美元的股权融资，其中包括300亿美元承销公开发行、400亿美元ATM发行计划，以及向Berkshire Hathaway私募出售100亿美元股票。公司说明资金用途包括扩张AI基础设施和全球算力，并披露AI需求超过当前可用供给。
- 关键数据或原文要点：官方公告披露，2026年CapEx预计为1800亿-1900亿美元，2027年CapEx预计较2026年“显著增加”；Google Cloud一季度收入同比增长63%，积压订单环比近乎翻倍至超过4600亿美元；过去12个月经营现金流1740亿美元，过去一年已通过多币种债务融资超过850亿美元。
- 来源链接：[Alphabet官方公告PDF](https://s206.q4cdn.com/479360582/files/doc_news/2026/Jun/01/attachments/2026-June-Alphabet-Equity-Capital-Raise-Press-Release-PDF.pdf)、[Reuters转引](https://www.investing.com/news/stock-market-news/alphabet-to-raise-80-billion-in-equity-capital-for-ai-spending-4720482)
- 对全球AI产业的影响：这是明确的超大规模云厂商AI CapEx上修与融资动作，说明AI算力需求仍处于供给约束状态。对GPU、TPU、服务器、网络、存储、电力、数据中心建设链条均构成中长期需求确认，但也提示AI基础设施资本强度继续上升，行业对外部融资和现金流承压的敏感度提高。
- 对A股映射方向的影响：偏利好AI基础设施映射方向，包括光模块/CPO、PCB、AI服务器、液冷、电力设备、IDC配套和半导体设备；同时需区分真正进入海外云厂商供应链、具备订单兑现能力的公司与仅主题映射公司。
- 影响等级：高
- 后续需要观察什么：融资是否按计划完成；2026-2027 CapEx执行节奏；Google Cloud backlog转收入速度；TPU与NVIDIA GPU采购结构；电力和数据中心建设是否成为瓶颈。

## 事件2
- 事件标题：BIS澄清对中国总部实体海外采购先进计算芯片仍需许可证
- 发生时间：2026-05-31至2026-06-01
- 涉及公司 / 产业：美国商务部BIS / NVIDIA / AMD / AI芯片出口管制 / 中国AI算力
- 事实摘要：美国商务部BIS发布指导，明确先进计算物项出口给总部位于Country Group D:5或澳门、或最终母公司位于上述地区的实体时，即使实体本身位于这些地区之外，也仍适用许可证要求。Reuters称该指导意在关闭中国AI企业通过海外子公司采购先进AI芯片的潜在漏洞。
- 关键数据或原文要点：BIS文件称许可证要求适用于所有美国以外目的地，前提是相关先进计算物项面向总部或最终母公司位于Country Group D:5或澳门的实体；同时，真实的数据中心运营商暂不需要因该指导停止已拥有先进计算物项的使用、存储、处置或服务。
- 来源链接：[BIS官方指导PDF](https://www.bis.gov/media/documents/bis-guidance-may-31-2026.pdf)、[Reuters转引](https://m.investing.com/news/stock-market-news/us-takes-step-to-halt-nvidia-ai-chip-shipments-to-chinese-firms-outside-china-4717939?ampMode=1)
- 对全球AI产业的影响：政策层面进一步收紧先进AI芯片跨境流向，可能降低中国企业通过第三地获取Blackwell等高端芯片的可行性。对全球非中国云厂商需求影响有限，但对中国AI训练算力、海外数据中心租赁与国产AI芯片替代逻辑影响较大。
- 对A股映射方向的影响：对国产AI芯片、国产服务器生态和算力租赁替代逻辑偏利好；对依赖海外高端GPU进口或海外转售路径的业务偏利空。由于BIS同时说明现有真实数据中心运营商无需立即停止使用或服务，短期存量影响仍需观察。
- 影响等级：高
- 后续需要观察什么：BIS后续是否扩大至数据中心服务和云算力租赁；NVIDIA/AMD中国相关收入指引是否变化；中国云厂商和模型公司是否加速国产芯片适配。

## 事件3
- 事件标题：NVIDIA称Vera Rubin进入量产爬坡，并确认AI CPU/GPU仍受供给约束
- 发生时间：2026-05-31至2026-06-02
- 涉及公司 / 产业：NVIDIA / AI服务器 / CPO网络 / GPU / AI数据中心
- 事实摘要：NVIDIA宣布Vera Rubin平台进入全量产爬坡，计划今年秋季开始生产出货；Reuters报道黄仁勋在台北表示公司已 확보足够供应以支持CPU和GPU的强劲增长，但仍处于供给受限状态，并称Vera CPU将成为新的主要增长驱动。
- 关键数据或原文要点：NVIDIA官方称Vera Rubin相对Grace Blackwell可提供10倍agent吞吐，涉及350多家工厂、30个国家和地区、台湾150家生态伙伴；Spectrum-X Ethernet Photonics已进入生产，面向百万GPU级AI工厂；Reuters报道黄仁勋称“仍然供给受限”。
- 来源链接：[NVIDIA官方公告](https://www.globenewswire.com/news-release/2026/06/01/3303979/0/en/nvidia-vera-rubin-ramps-into-full-production-to-power-agentic-ai-factories-worldwide.html)、[Reuters转引](https://www.streetinsider.com/Reuters/Nvidia%2BCEO%2Bsays%2Bhas%2Bcapacity%2Bto%2Bsupply%2Brobust%2BCPU%2Band%2BGPU%2Bgrowth/26585767.html)
- 对全球AI产业的影响：说明NVIDIA路线图继续按AI工厂、机柜级/POD级系统推进，价值量从单GPU扩展到CPU、网络、存储、安全和整机柜系统。供给约束表述强化AI需求仍高于有效供给，但后续需要用订单、出货和数据中心收入兑现来验证。
- 对A股映射方向的影响：偏利好光模块/CPO、AI服务器、PCB、液冷、连接器、电源和高速网络相关方向；Vera CPU增长叙事也会提高市场对AI服务器CPU、内存和整机系统价值量的关注。
- 影响等级：中
- 后续需要观察什么：Vera Rubin秋季生产出货是否按期；GB200/GB300到Rubin过渡是否造成供应链拉货波动；CPO从样机/验证到规模出货的节奏；HBM4供应和先进封装产能。
