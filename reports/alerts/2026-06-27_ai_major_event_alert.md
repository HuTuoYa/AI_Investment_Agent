## 事件1
- 事件标题：AWS 7月将更新 EC2 Capacity Blocks for ML 价格，媒体称多类AI GPU预留算力价格约上调20%
- 发生时间：2026-06-26（媒体发布时间）；AWS页面显示当前价格计划在2026年7月更新
- 涉及公司 / 产业：Amazon / AWS；AI云算力、NVIDIA GPU、HBM、数据中心、AI基础设施成本
- 事实摘要：AWS官方定价页显示，EC2 Capacity Blocks for ML 的预留价格会根据供需趋势定期更新，当前价格计划在2026年7月更新；该服务覆盖NVIDIA Blackwell B200、H200、H100、A100以及AWS Trainium等加速实例。Business Insider报道称，AWS多类AI云服务器小时费率将在7月起约上调20%，并将原因指向内存/HBM成本上升和GPU供给紧张；该媒体还提到AWS在2026年1月已对同一服务上调约15%。
- 关键数据或原文要点：
  - AWS官方：EC2 Capacity Blocks for ML可预留Amazon EC2 UltraClusters中的加速计算实例，支持P6e-GB200、P6-B300、P6-B200、P5en、P5e、P5、P4d、Trn2、Trn1等实例。
  - AWS官方：预留价格会基于EC2 Capacity Blocks供需趋势定期更新，当前价格计划在2026年7月更新。
  - 媒体报道：多类AI云服务器小时费率约上调20%；报道将涨价与HBM/内存成本和供给紧张联系起来。由于AWS官方页面未直接披露“约20%”涨幅，具体涨幅仍以AWS实际生效价格为准。
- 来源链接：
  - AWS官方产品页：https://aws.amazon.com/ec2/capacityblocks/
  - AWS官方定价页：https://aws.amazon.com/ec2/capacityblocks/pricing/
  - Business Insider报道：https://www.businessinsider.com/amazon-raises-ai-cloud-prices-memory-chip-costs-soar-2026-6
- 对全球AI产业的影响：这是供需偏紧和AI算力成本上行的直接信号。它不等同于AWS上调全年CapEx，也不是NVIDIA财报层面的新增指引，但它说明高端GPU/HBM/数据中心资源仍具定价权，AI训练和推理企业的算力成本可能继续承压。对云厂商而言，若客户仍接受涨价，说明AI算力需求韧性较强；若后续需求被价格抑制，则需要警惕AI应用商业模式和算力消耗效率的压力。
- 对A股映射方向的影响：偏利好上游AI基础设施链条，尤其是与高端GPU服务器、HBM配套、光模块、PCB、液冷、电力设备和数据中心供给相关的方向；但对下游AI应用公司偏中性到压力，因为云端推理/训练成本上升可能压缩利润率。对A股映射不宜直接外推为订单确认，仍需观察国内相关公司公告、订单和毛利率。
- 影响等级：中
- 后续需要观察什么：
  - AWS 7月实际生效价格表是否确认约20%涨幅，以及涉及P6/B200、P5e/H200、P5/H100的范围。
  - Microsoft Azure、Google Cloud、Oracle Cloud是否跟随上调GPU预留/短租价格。
  - HBM价格、SK hynix/Samsung/Micron供给指引，以及NVIDIA Blackwell出货节奏是否继续受HBM或封装约束。
  - AI客户是否因价格上升放缓训练/推理扩容，或转向自研芯片、Trainium/TPU等替代方案。

## 事件2
- 事件标题：美国两党议员拟推出Cloud Security Act，关注通过云服务绕开AI芯片出口管制的风险
- 发生时间：2026-06-26（媒体发布时间）
- 涉及公司 / 产业：美国云厂商、AI云服务、先进AI芯片出口管制、NVIDIA/AMD GPU远程使用
- 事实摘要：Axios报道称，美国众议员Josh Gottheimer和John Moolenaar拟推出Cloud Security Act，允许美国云服务商向商务部报告外国实体疑似滥用美国先进AI算力的行为，目标是补上通过美国云计算产品访问先进芯片的潜在监管漏洞。该法案仍处于提出阶段，尚未成为法律。
- 关键数据或原文要点：报道核心在于把AI芯片出口管制从“实体芯片流向”进一步延伸到“云端远程算力使用”的监管方向；目前缺少正式通过后的执行细则，影响力度不足以判断。
- 来源链接：
  - Axios报道：https://www.axios.com/2026/06/26/gottheimer-moolenaar-ai-cloud-security-bill
- 对全球AI产业的影响：如果后续立法推进并形成强制性规则，可能提高美国云厂商对跨境AI算力客户的合规审查成本，并影响部分海外客户获取高端GPU云算力的便利性。当前仍是政策意向信号，不足以判断会明显改变短期供需。
- 对A股映射方向的影响：对国产AI芯片、国产算力租赁和本土云算力替代逻辑偏正面，但短期没有订单或业绩验证；对依赖海外云端高端GPU的国内AI应用或模型训练主体可能偏约束。
- 影响等级：低
- 后续需要观察什么：
  - 法案是否正式提交、进入委员会审议、获得两院通过。
  - 商务部是否配套出台KYC、报告义务、客户限制或算力门槛规则。
  - 云厂商是否调整境外客户GPU云服务准入、定价或区域可用性。
