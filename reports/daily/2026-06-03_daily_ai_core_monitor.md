# AI核心指标日报：2026-06-03

## 1. 今日结论

过去24小时最重要的新信号来自Alphabet：公司宣布拟进行合计800亿美元股权融资，用于扩张AI compute基础设施，并披露2026年CapEx预计为1800-1900亿美元、2027年CapEx将较2026年显著增加。NVIDIA方面，未发现新的财报或正式指引更新，但Reuters报道Jensen Huang在Computex/GTC Taipei期间表示公司已为CPU/GPU系统锁定“very robust growth”所需供应，同时仍处于供给受限状态；这与其5月20日Q1 FY2027财报中数据中心收入752亿美元、同比增长92%的强需求背景一致。HBM侧，过去24小时有SK Group/Reuters关于SK hynix五年内计划翻倍晶圆产能的报道，指向AI内存供给仍是关键瓶颈。A股映射上，海外AI基础设施强投入继续偏利好光模块、PCB、AI服务器、液冷/电力和数据中心方向，但也需要警惕估值与订单兑现节奏。

- 全球AI CapEx：增强
- NVIDIA需求信号：增强
- A股AI基础设施映射：偏利好

## 2. 全球AI CapEx更新

### Alphabet / Google

- 事件：Alphabet于2026-06-01宣布拟进行合计800亿美元股权融资，称资金用途包括扩张AI基础设施和全球算力，以应对“unprecedented customer demand”。官方文件披露：300亿美元承销公开发行、400亿美元ATM计划、100亿美元Berkshire Hathaway私募投资；2026年CapEx预计1800-1900亿美元，2027年CapEx预计较2026年显著增加。
- 关键数据：Q1 2026收入1100亿美元，同比增长22%；Google Cloud收入同比增长63%，backlog环比接近翻倍至超过4600亿美元，约50%预计在未来24个月确认收入。
- 来源：Alphabet官方公告PDF：https://s206.q4cdn.com/479360582/files/doc_news/2026/Jun/01/attachments/2026-June-Alphabet-Equity-Capital-Raise-Press-Release-PDF.pdf；Axios补充报道：https://www.axios.com/2026/06/01/alphabet-80-billion-ai-buildout；Semafor补充报道：https://www.semafor.com/article/06/02/2026/alphabet-to-sell-80-billion-in-stock-to-fund-ai-buildout
- 产业链含义：这是明确的AI CapEx强化信号。融资用途直接指向AI compute和全球基础设施，利好GPU、HBM、光通信、交换网络、电力、液冷、AI服务器和IDC建设链条。需要注意ATM中约300亿美元主要用于员工股权税务义务安排，不能全部等同新增AI CapEx。

### Microsoft / Amazon AWS / Meta / Oracle / Tesla

- 过去24小时未发现这些公司发布新的官方AI CapEx上修/下修、数据中心建设指引或财报更新。继续跟踪公司IR、财报和电话会页面：Microsoft IR https://www.microsoft.com/en-us/Investor；Amazon IR https://ir.aboutamazon.com/；Meta IR https://investor.fb.com/；Oracle IR https://investor.oracle.com/；Tesla IR https://ir.tesla.com/
- 含义：除Alphabet外，其他主要云厂商暂无新增正式CapEx变化，行业判断仍以此前“高位投入”延续为主，不足以判断出现新的集体上修或下修。

## 3. NVIDIA更新

### NVIDIA供给与需求表述

- 事件：Reuters于2026-06-02报道，Jensen Huang在台北Computex/GTC Taipei期间表示，NVIDIA已有足够供应支持CPU和GPU系统的强劲增长，但公司仍处供给受限状态。
- 涉及指标：GPU需求、CPU/GPU系统供应、订单可见度、供给瓶颈。
- 来源：Reuters转载页：https://www.streetinsider.com/Reuters/Nvidia%2Bhas%2Bcapacity%2Bto%2Bsupply%2Brobust%2BAI%2Bgrowth%2Bdespite%2Bconstraints%2C%2Bsays%2BCEO/26585767.html
- 含义：需求信号偏强，供给约束仍是收入确认与下游交付节奏的关键变量；对光模块、PCB、AI服务器、液冷、电力设备、HBM和先进封装链条偏正面。

### NVIDIA最新官方财务基准

- 事件：NVIDIA于2026-05-20发布Q1 FY2027财报，收入816亿美元，同比增长85%、环比增长20%；数据中心收入752亿美元，同比增长92%、环比增长21%；Q2 FY2027收入指引为910亿美元正负2%，且未假设来自中国的数据中心计算收入。
- 涉及指标：收入、数据中心收入、指引、Blackwell/AI factory需求、出口限制。
- 来源：NVIDIA投资者关系公告：https://investor.nvidia.com/news/press-release-details/2026/NVIDIA-Announces-Financial-Results-for-First-Quarter-Fiscal-2027/default.aspx
- 含义：财报基准仍指向AI数据中心需求高景气，但中国数据中心收入未纳入指引，出口限制仍是重要不确定性。

### Vera Rubin / Blackwell / HBM与系统生态

- 事件：NVIDIA官方5月31日宣布Vera Rubin平台进入全量生产，供应链覆盖台湾150家生态伙伴、350多家工厂和30个国家，并推出Spectrum-X Ethernet Photonics用于百万GPU级AI factories。该事件略早于本次24小时窗口，但与6月2日管理层表述互相印证。
- 涉及指标：下一代AI服务器、网络、光通信、液冷和全球供应链。
- 来源：NVIDIA官方公告：https://nvidianews.nvidia.com/news/vera-rubin-full-production-agentic-ai-factory
- 含义：NVIDIA平台从GPU扩展到CPU、网络、存储、光互连和整机系统，A股映射不应只看GPU服务器，还要跟踪光模块/光器件、交换网络、液冷、电力与数据中心建设。

### HBM供应

- 事件：Reuters于2026-06-02报道，SK Group董事长称SK hynix计划未来五年将晶圆产能翻倍；报道同时提到SK hynix是NVIDIA HBM主要供应商，Counterpoint数据显示其一季度全球HBM份额为58%，Samsung与Micron各为21%。
- 来源：Reuters转载页：https://www.marketscreener.com/news/sk-hynix-plans-to-double-wafer-capacity-in-next-five-years-group-chairman-says-ce7f5dd9d180f721
- 含义：HBM供给仍是AI GPU放量关键约束。扩产计划偏长期，短期仍需关注HBM4良率、交付节奏、先进封装和GPU系统出货。

## 4. A股映射方向

- 光模块：强。Alphabet融资和NVIDIA百万GPU AI factory/光互连叙事强化800G、1.6T、CPO、硅光需求逻辑，但需验证海外大客户订单和毛利率。
- PCB：较强。AI服务器、GPU/NVL rack、交换机和高速互连升级继续提升高多层、高频高速PCB需求。
- AI服务器：较强。NVIDIA供给仍受限但已锁定强增长供应，整机、ODM、液冷机柜和电源链条景气延续。
- 液冷/电力：较强。AI factory功率密度持续上升，液冷、UPS、变压器、储能、数据中心温控和电网接入仍是重点。
- 数据中心/算力租赁：中性偏强。海外CapEx强度支持行业需求，但国内算力租赁需要重点核验利用率、客户质量、回款和融资成本。
- 半导体设备/国产AI芯片：中性偏强。HBM与先进封装扩产、出口限制和国产替代逻辑仍在，但业绩兑现周期较长。
- 机器人/自动驾驶：中性。NVIDIA Edge Computing框架与物理AI继续推进，但本次24小时窗口未发现Tesla或自动驾驶方向重大新增CapEx/订单信息。

## 5. 风险提示

- 暂未发现过去24小时主要云厂商集体下修AI CapEx的明确信号。
- 未发现NVIDIA新收入或指引不及预期信息；相反，最新官方财报与管理层表述仍偏强。
- 风险在于：AI数据中心投资过热、融资依赖上升、CapEx回报周期拉长；HBM、先进封装、光通信和电力设备供给瓶颈导致交付节奏波动；美国出口限制使中国相关收入不确定；A股AI基础设施板块估值可能先于订单和利润兑现。
- 对传闻或非官方供应链消息不应直接当作确定订单，需要继续回到财报、公告、电话会和客户验收数据验证。

## 6. 需要重点关注的问题

1. Alphabet 800亿美元融资完成节奏、2026年1800-1900亿美元CapEx是否继续上修，以及Google Cloud backlog能否按期转化为收入。
2. NVIDIA Q2 FY2027实际收入、数据中心收入、毛利率与中国数据中心收入假设是否变化，Blackwell/Vera Rubin交付是否受HBM或先进封装限制。
3. A股光模块、PCB、AI服务器、液冷和电力设备公司能否把海外AI CapEx转化为可验证订单、收入、毛利率和现金流，而非仅停留在主题估值。
