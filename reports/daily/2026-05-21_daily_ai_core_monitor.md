## 3. NVIDIA更新

### 事件：FY2027 Q1 财报发布，数据中心收入和 FY2027 Q2 指引继续上行

- 关键数据：NVIDIA FY2027 Q1 收入 816.15 亿美元，环比增长 20%，同比增长 85%；GAAP 毛利率 74.9%，Non-GAAP 毛利率 75.0%。来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-first-quarter-fiscal-2027)。
- 数据中心：Data Center 收入 752 亿美元，环比增长 21%，同比增长 92%。在此前分项口径下，Data Center compute 收入 604 亿美元，同比 +77%，环比 +18%；Data Center networking 收入 148 亿美元，同比 +199%，环比 +35%。来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-first-quarter-fiscal-2027)。
- 指引：FY2027 Q2 收入指引为 910 亿美元，上下浮动 2%；公司明确说明该指引未假设来自中国的数据中心计算收入。来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-first-quarter-fiscal-2027)。
- Blackwell/推理：NVIDIA 称 Dynamo 1.0 已进入生产，可在 Blackwell GPU 上提升生成式与 agentic inference；并披露 Google Cloud 将提供 Vera Rubin-powered A5X instances，同时在 Google Distributed Cloud 上预览运行于 Blackwell/Blackwell Ultra GPU 的 Gemini 模型。来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-first-quarter-fiscal-2027)。
- 光通信/网络：公司宣布与 Marvell 在 NVLink Fusion 与硅光技术方面合作，并与 Coherent、Corning、Lumentum 达成多年战略协议以推进先进光学。来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-first-quarter-fiscal-2027)。
- 市场预期对比：Reuters 报道，NVIDIA FY2027 Q1 收入 816.2 亿美元，高于 LSEG 汇总的分析师平均预期 788.6 亿美元；同时公司给出高于预期的下一季度收入预测，并宣布新增 800 亿美元回购授权。来源：[Reuters via Investing.com](https://www.investing.com/news/stock-market-news/nvidia-forecasts-quarterly-revenue-above-estimates-announces-80-billion-share-buyback-4702363)。
- 产业含义：数据中心 compute 与 networking 同步创新高，说明 AI 训练、推理、网络互联和数据中心内部光互联仍处在扩张阶段。Blackwell 与 Rubin 相关合作强化了先进 GPU、HBM、AI服务器、交换网络、光模块、PCB、液冷与电力配套的景气逻辑。

## 4. A股映射方向

- 光模块：影响偏强。NVIDIA 数据中心 networking 收入同比 +199%、环比 +35%，并宣布与 Coherent、Corning、Lumentum 的先进光学合作，对 800G/1.6T、硅光、CPO 等方向构成景气验证。
- PCB：偏利好。Blackwell/Rubin 平台、AI服务器和高速互联升级继续提升高多层、高速、高可靠 PCB 需求，但需关注 A股公司订单兑现与毛利率。
- AI服务器：偏利好。数据中心收入和 FY2027 Q2 指引强于预期，说明整机、ODM、电源、散热、结构件仍有需求支撑。
- 液冷/电力：偏利好。AI factory 与高功率机柜扩张继续推升液冷、UPS、配电、电网侧扩容与数据中心温控需求。
- 数据中心/算力租赁：中性偏利好。需求端强，但需警惕算力租赁企业资产负债表、客户质量、GPU折旧和租金下行风险。
- 半导体/国产AI芯片：结构性利好。NVIDIA 指引不包含中国数据中心计算收入，意味着出口限制仍可能推动国产替代叙事；但国产芯片的商业化、生态和交付仍需独立验证。
- 机器人/自动驾驶：中性偏利好。NVIDIA Edge Computing 收入同比 +29%，且披露 DRIVE、Cosmos、Isaac、Thor 等进展，对机器人与自动驾驶主题有映射，但今日主线仍是数据中心而非物理 AI。

## 5. 风险提示

- AI CapEx下修：今日未发现主要云厂商新增下修，但如果后续 Microsoft/Amazon/Alphabet/Meta/Oracle 对 CapEx 做降温表述，光模块、AI服务器、PCB、数据中心链条可能承压。
- NVIDIA收入或指引不及预期：今日未出现，反而收入与指引强于市场预期；但高基数下未来增速放缓风险仍需跟踪。
- GPU需求放缓：今日数据不足以支持需求放缓判断；但推理芯片竞争、自研 ASIC、客户资本效率审查仍是中期风险。
- 出口限制：中国数据中心计算收入未纳入 NVIDIA FY2027 Q2 指引，是明确风险项，也可能导致全球供需和中国本土替代路径分化。
- HBM/先进封装/供应链：HBM 与 CoWoS 等瓶颈若缓解不及预期，会限制 GPU 出货；若快速扩产，则可能带来价格和库存周期风险。
- 数据中心投资过热：AI factory 扩张速度快，需警惕电力、土地、回报周期、客户集中度与债务融资压力。
- A股估值风险：光模块、PCB、液冷、算力租赁、机器人等方向若估值先行过快，后续订单或业绩兑现不及预期会放大波动。
